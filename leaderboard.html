🔗 Game Integration Guide

How to connect your games to the leaderboard:

1. At the end of each game, call: submitScore(gameName, playerScore)

2. Game names should match exactly: "2048", "Snake", "Cookie Clicker", etc.

3. Player will be prompted to enter their name

4. Scores are automatically ranked and displayed here

Example: submitScore("Snake", 2850);

Supported Games: 2048, Animal Twenty Questions, Brick Breaker, Chess, Connect 4, Cookie Clicker, Dodge, Emoji Guess, Evolmon, Hangman, Imposter, Memory Cards, Pong, Snake, Tic-Tac-Toe, Trivia, Typing Speed, and many more!
